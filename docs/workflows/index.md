# Workflows

Workflows show how to use DiffOrb for real orbit-determination and ephemeris-generation tasks. They connect data loading, model configuration, propagation, fitting, and result inspection into complete runs.

[Guides](../guides/index.md) show concrete tasks. [Concepts](../concepts/index.md) explain the models behind each step.

## Available Workflows

- [Ephemeris Products Workflow](ephemeris-products-workflow.md): propagate one body once, build ephemeris products, and compare optical output with JPL Horizons.
- [Orbit Determination Workflow](orbit-determination-workflow.md): load online observations, fit an orbit, inspect station weights, and rerun the fit.

## When To Use Workflows

- Use these workflows when you need to see a full run from start to finish.
- Use [Guides](../guides/index.md) when you need one operation.
- Use [Concepts](../concepts/index.md) when you need the model behind a step.
- Use [API](../api/index.md) when you need exact arguments and return fields.
