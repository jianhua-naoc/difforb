# API Scope And Conventions

These pages document the public APIs exported by top-level `difforb` subpackages and selected nested modules with their own public surface.

## Included

- top-level subpackages exported by `difforb.__all__`,
- selected nested public modules such as `difforb.core.eop`,
- interfaces exported by each subpackage's `__all__`,
- object documentation for those exported interfaces only.
